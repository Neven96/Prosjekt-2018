For Windows:

1. Last ned NodeJS her: "https://nodejs.org/en/" og installer den

2. Åpne din terminal of choice, CMD eller PowerShell, og naviger til mappen hvor denne filen ligger

3. Bruk kommandoen "npm install" for å installere NodeJS i mappen

4. Bruk kommandoen "npm start" for å starte programmet

For Mac:

1. Last nede NodeJS her: "https://nodejs.org/en/" og installer den

2. Åpne terminalen og naviger til mappen hvor denne filen ligger

3. Bruk kommandoen "npm install" for å installere NodeJS i mappen

4. Bruk kommandoen "npm start" for å starte programmet

For Linux Ubuntu:

1. Installer NPM via kommandoen "sudo apt install npm" i terminalen

2. Naviger til mappen hvor denn filen ligger

3. Bruk kommandoen "npm install" for å installere NodeJS i mappen

4. Bruk kommandoen "npm start" for å starte programmet
